Untuk mengaktifkan ekstentsi tersebut, melalu XAMPP Control Panel, pada bagian Apache 
klik Config -> "PHP.ini"

setelah anda masuk kebagian tab "php.ini", ada Beberapa ekstensi PHP yang perlu anda aktifkan untuk kebutuhan pengembangan 
Codeigniter 4. diantaranya
• php-json ekstension untuk bekerja dengan JSON; 
• php-mysqlnd native driver untuk MySQL; 
• php-xml ekstension untuk bekerja dengan XML; 
• php-intl ekstensi untuk membuat aplikasi multibahasa; 
• libcurl (opsional), jika ingin pakai Curl.
